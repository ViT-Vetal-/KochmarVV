Все не авторизованные пользователи попадают на страницу First_Reg.aspx, на которой есть форма входа.
Стандартное имя для входа newcomer, а пароль 111111 (шесть единиц). 

Далее регистрируемся и попадаем на страницу входа. Вы уже "внутри", go, go, go!:-)

Попадаем на Календарь. Пока там пусто!!! Переходим в раздел "Добавить запись".

При добавлении записи первым делом выбираем день (просто тыкнув на него в календаре!!!) иначе запись будет без даты!  Далее вводим необходимые нам данные. Возвращаемся в раздел "Календарь".

Теперь давим на день в календаре (в который добавили нашу запись) и видим, что ниже календарика появилась табличка с данными.

P.S.Каждый пользователь может видеть только свои данные!!!
